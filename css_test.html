CSS Display Properties: Block, Inline, Inline-Block

  • The following div is generated by JS. It has 3 nested loops each having a class called outer, middle, inner.
  • The classes decrese in font-size, border and padding as we go deep. The inner boxes do not have border. Margins don't exist on any box.
  • Each loop runs for 3 iterations named block, inline, inline-block. It corresponds to the display property the class sets to the box.
